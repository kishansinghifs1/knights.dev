# Kinght.dev

Welcome to the **Kinght.dev** project!

## Initial Setup

1. **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/kinght.dev.git
    cd kinght.dev
    ```

2. **Install dependencies:**
    ```bash
    # For npm
    npm install

    # Or for yarn
    yarn install
    ```

3. **Start the development server:**
    ```bash
    npm start
    # or
    yarn start
    ```

## Project Structure

- `src/` - Source code
- `public/` - Static assets
